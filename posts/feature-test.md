# 마크다운 기능 테스트

이 포스트는 블로그의 마크다운 파서가 지원하는 **모든 기능**을 한 번에 확인하기 위한 테스트 문서입니다. 목차, 읽기 진행바, 다크 모드, 스크롤 하이라이트도 함께 확인할 수 있습니다.

## 텍스트 강조

일반 텍스트, **굵은 글씨**, *기울임꼴*, 그리고 ***굵은 기울임꼴***을 섞어 쓸 수 있습니다. 밑줄 문법으로도 __굵게__ 와 _기울임_ 이 됩니다. 문장 중간의 `인라인 코드`도 잘 렌더링되는지 확인합니다.

## 코드 블록

언어 라벨이 우측 상단에 표시되는지 확인합니다.

```javascript
async function loadPost(slug) {
  const res = await fetch(`posts/${slug}.md`);
  if (!res.ok) throw new Error('not found');
  return await res.text();
}
```

```python
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
```
